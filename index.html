Graph Project

Tile Based Game

Описание проекта

Этот проект представляет собой игру на основе плиток, где игрок управляет големом, который должен пройти через лабиринт, собирая монеты и избегая врагов и ловушек. Игра включает в себя несколько комнат, каждая из которых становится сложнее по мере прохождения. Игрок может использовать различные способности и улучшения, чтобы помочь себе в прохождении.

Инструкция по запуску

  1. Убедитесь, что у вас установлен Python 3.x и библиотека Pygame.
  2. Скачайте исходный код проекта из репозитория.
  3. Перейдите в директорию с проектом и запустите файл main.py.
  4. Следуйте инструкциям на экране, чтобы начать игру.
